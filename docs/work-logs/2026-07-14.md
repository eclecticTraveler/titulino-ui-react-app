**Extension pattern for future external link actions:**
1. Add JSON entry in `floating-actions.json` (`type: "link"` or `type: "resolver"`)
2. Add entry to `ACTION_ICONS` map in `FloatingActionMenu/index.js`
3. Add LESS modifier `&--{id} { background-color: ...; color: ...; }`
4. Add localization key `floating.{id}.tooltip`
