### CourseFeatureConfig â€” per-course feature flags (new file)
Created `src/configs/CourseFeatureConfig.js` with a static per-theme feature flag registry.

```js
export const COURSE_FEATURE_FLAGS = {
    meditaciones: {
        hasGrammarClass: false,
        hasQuizletAndExercises: false,
    }
};
export const getCourseFeatureFlags = (theme) => ({
    hasGrammarClass: true,
    hasQuizletAndExercises: true,
    ...(COURSE_FEATURE_FLAGS[theme] || {})
});
```

- `getCourseFeatureFlags(theme)` spreads per-course overrides onto a true-by-default baseline â€” all existing callers (no argument) are unaffected.
- Adding flags for a future course = one entry here, zero other files touched.
