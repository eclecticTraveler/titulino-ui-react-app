## Architecture decisions recorded in plan

- `coursePath` stored in `CourseDetails` JSONB per course instance, NOT in `course-theme-registry.data.json` (spine bucket would break `buildCodeToTheme` which calls `.forEach` on every registry value).
- Authenticated hub fetches courses locally to avoid polluting `lrn.availableCourses` Redux state (enrollment flow uses that and filters enrolled courses out).
