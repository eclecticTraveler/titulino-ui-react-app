### TitulinoWorkerService — GitHub Actions YAML fix

**File:** `TitulinoWorkerService/.github/workflows/deploy-titulino-worker.yml` (line 68)

The "Add version file" step used a bare shell redirect (`>`), which YAML interprets as a folded-block scalar indicator, causing a workflow parse error.

```yaml
# Before (broken — YAML parses > as folded scalar)
run: echo "Build: $(date +%Y%m%d_%H%M%S)" > ./publish/version.txt

# After (fixed — shell command runs inside | block scalar)
run: |
  echo "Build: $(date +%Y%m%d_%H%M%S)" > ./publish/version.txt
```

Commit the workflow file to `TitulinoWorkerService` to clear the GitHub Actions parse error.
