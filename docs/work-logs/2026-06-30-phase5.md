### feedback_json shape (T33)

Before T33, `feedback_json` DB column stored: `{ "q2_bio": "summary text..." }`

After T33 it stores:
```json
{
  "q2_bio": {
    "summary": "The main corrections were...",
    "grammarNotes": ["Changed 'go' to 'went' (Simple Past required)"],
    "vocabularySuggestions": ["Replace 'good' with 'excellent' for stronger impact"]
  }
}
```

No DB migration needed — `feedback_json` is a jsonb column and `KnowMeAiResultResponse.Feedback` is `JsonElement?` so the richer shape flows through to the React frontend as-is.
